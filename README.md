# Artisan Bakery Project 🥐

A beautiful and responsive multi-page website for a professional bakery. This project showcases a variety of bakery products, a gallery, and a custom menu with a modern user interface.

## ✨ Features
- **Multi-page Layout**: Includes Home, About, Menu, and Gallery pages.
- **Responsive Design**: Optimized for all devices (Mobile, Tablet, and Desktop).
- **Interactive Gallery**: High-quality images showcasing bakery items.
- **Clean Code**: Semantic HTML5 and modular CSS3 structure.

## 🛠 Tech Stack
- **HTML5**: For structured web content.
- **CSS3**: Custom styling and layout design.
- **Google Fonts**: For professional typography.
